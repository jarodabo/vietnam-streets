ðŸ”§ Advanced Usage

JavaScript API

Create embeds programmatically using JavaScript:

// Create a story embed const storyEmbed = VietnamStreets.createStoryEmbed({ width: '100%', storyId: 'story-1' // Use specific story IDs }); // Create a stories grid const gridEmbed = VietnamStreets.createStoriesGridEmbed({ width: '100%', columns: 3, limit: 6, category: 'heritage' // Filter by category }); // Create photographer showcase const photoEmbed = VietnamStreets.createPhotographerEmbed({ width: '500px', limit: 3 }); // Create events calendar const eventsEmbed = VietnamStreets.createEventsEmbed({ width: '100%', limit: 3 }); // Add to your page document.getElementById('my-container').appendChild(storyEmbed);

Available Configuration Options

All embed types support these configuration options:

{ width: '100%', // CSS width value height: 'auto', // CSS height value (optional) // Story embed specific storyId: 'latest', // 'latest', 'story-1', 'story-2', etc. // Grid embed specific columns: 3, // 2 or 3 columns limit: 6, // Number of items to show category: 'heritage', // Filter by category (optional) // Photographer embed specific limit: 3, // Number of photographers // Events embed specific limit: 3 // Number of events }

Event Callbacks

Listen for embed events:

// Listen for embed ready events document.addEventListener('vietnam-streets-embed-ready', function(e) { console.log('Embed loaded:', e.detail); }); // Check if VietnamStreets is loaded if (window.VietnamStreets) { console.log('Vietnam Streets embed version:', VietnamStreets.version); }

Data Sources

The embed system loads data from multiple sources:

  • Local JSON files: ../data/stories.json, ../data/photographers.json, ../data/events.json
  • API fallback: vietnamstreets.com/api/* endpoints
  • Demo data: Hardcoded fallback for offline demos

ðŸ”§ Troubleshooting

Common Issues

Embed not showing up:

  • Check that the script tag is properly included in your HTML head
  • Verify the embed div has the correct data attributes
  • Check browser console for JavaScript errors

Data not loading:

  • Check network tab for failed API requests
  • Verify CORS settings if serving from different domain
  • Fallback demo data should still display

Styling issues:

  • Use CSS specificity to override embed styles
  • Add !important to ensure your styles take precedence
  • Check for conflicting CSS that might affect layout

Browser Support

Vietnam Streets embeds work in all modern browsers:

  • Chrome 60+
  • Firefox 55+
  • Safari 12+
  • Edge 79+

Performance

The embed system is designed for performance:

  • Lightweight JavaScript (~15KB gzipped)
  • CSS injected only once per page
  • Lazy loading for images and content
  • Graceful fallbacks for offline scenarios
